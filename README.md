# blind-grid-run
A puzzle game where you navigate to the goal, but with a twist - you have limited sight!

## Features
- Multiple levels
- Limited visibility system
- Smooth movement using lerp
- Sound effects and level transitions

## Controls
- Arrow keys to move

## Technology Used
- JavaScript
- p5.js
